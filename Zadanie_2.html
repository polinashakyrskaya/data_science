Сессия_1¶

  • Разведочный анализ данных.
  • Обработка пропущенных значений.
  • Кодирование всех категориальных признаков.
  • Нормализация данных.
  • Преобразование значений.

Выполнения этих пунктов поспосопствует хорошему обучению.

Обработка пропущенных значений¶

Существует три способа обработки пропущенных значений количественных признаков, и два способа для категориальных признаков:

Для количественных признаков:

  • Удаление значений или признаков с пропущенными значениями(используется если размер выборки большой, а пропущенных значений мало.)
  • Замена пропущенных значений на статистические велечины(используется если значение отсутсвует по каким-то случайным ситуациям, считается самым стандартным способом.)
  • Прогнозирование значений(приеняется если значения отсутствуют не случайно, и имеют взаимосвязь между переменными выбранами для вминения, иначе это может привести к неточной оценки.)

Для категориальных признаков:

  • Замнена пропущенных значений на моду числа(хорошо работает, если очень много повторяющихся признаков.)
  • Замнена пропущенных значений на новую категорию(если значение отсутствует случайно.)

Для того, что-бы определить каким методом воспльзоваться, нужно вывести процент пропущенных значений для каждого атрибута, если атрибут имеет больше 80 процентов, он считается неинформативным и его необходимо удалить:

for col in df.columns:
    missing=np.mean(df[col].isnull())
    missing_non=missing*100
    print('Признак:{}-пропущенные значения:{}%'.format(col, round(missing_non)))
    if missing_non>80:
        del df[col]
